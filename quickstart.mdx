---
title: "Quickstart"
description: "Compile and run the Family Tree program in minutes"
---

<Warning>
  The file chooser is a GUI dialog — running in a headless environment (no display) will fail with a `HeadlessException`. Make sure you have a graphical display available before launching the program.
</Warning>

<Steps>

<Step title="Prerequisites">
  Java JDK 8 or later is required. Swing is bundled with the JDK — no external libraries or build tools are needed.

  Verify your Java version:

  ```bash
  java -version
  ```

  You should see output like `openjdk version "11.0.x"` or later.
</Step>

<Step title="Clone the repository">
  ```bash
  git clone https://github.com/ehcaw/lab12.git
  cd lab12
  ```
</Step>

<Step title="Compile the source">
  Create a `bin/` output directory and compile both source files together:

  ```bash
  mkdir -p bin
  javac -d bin src/trees/FamilyTree.java src/trees/TreeException.java
  ```

  If compilation succeeds, `bin/trees/FamilyTree.class` and `bin/trees/TreeException.class` will be created.
</Step>

<Step title="Create a sample input file">
  Create a `data/` directory and add a sample tree file:

  ```bash
  mkdir data
  ```

  Create `data/tolkien.txt` with the following content:

  ```text tolkien.txt
  Thror:Thrain
  Thrain:Thorin,Frerin,Dis
  Thorin:Fili,Kili
  Dis:Bilbo
  Bilbo:Frodo
  ```

  Each line follows the format `parent:child1,child2,...`. The first line in the file establishes the root of the tree. Every subsequent parent must already exist as a child from an earlier line.

  <Tip>
    The `data/` directory is checked first by convention. If it does not exist, the file chooser dialog opens in the current working directory instead.
  </Tip>
</Step>

<Step title="Run the program">
  ```bash
  java -cp bin trees.FamilyTree
  ```

  A Swing file chooser dialog opens. Navigate to and select `tolkien.txt`. After you confirm, the program prints the full tree followed by the most recent common ancestor of `Bilbo` and `Frodo`.

  <CodeGroup>

  ```text expected output
  Tree:
  Family Tree:

  Thror
    Thrain
      Thorin
        Fili
        Kili
      Frerin
      Dis
        Bilbo
          Frodo


  **************

  Most recent common ancestor of Bilbo and Frodo is Dis
  ```

  </CodeGroup>

  The MRCA result shows `Dis` because `Dis` is the parent of `Bilbo`, who is the parent of `Frodo` — making `Dis` the deepest ancestor shared by both.
</Step>

<Step title="Try your own tree">
  Create any `.txt` file following the same line format:

  ```text
  parent:child1,child2,...
  ```

  Rules to keep in mind:

  - The very first line defines the root node.
  - Every parent named on subsequent lines must have already appeared as a child on an earlier line.
  - A parent can have any number of comma-separated children on a single line.
  - Names are case-sensitive: `Bilbo` and `bilbo` are treated as different individuals.

  Run the program again with `java -cp bin trees.FamilyTree` and select your new file when the dialog opens.
</Step>

</Steps>
