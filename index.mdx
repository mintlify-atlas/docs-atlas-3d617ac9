## Overview

Lab 12 implements a **family tree** data structure in Java. It reads a structured text file describing parent-child relationships, builds an in-memory tree, and provides methods to traverse ancestry chains and find the **most recent common ancestor (MRCA)** between any two individuals.

## How it works

The tree is built from a plain text file where each line describes one parent and its immediate children:

```text family-tree.txt
GrandparentA:ParentA,ParentB
ParentA:Child1,Child2
ParentB:Child3
```

The `FamilyTree` class parses this file line by line, creates `TreeNode` objects, and links them via parent and children references. Once built, you can call `getMostRecentCommonAncestor` to find the deepest shared ancestor of any two people in the tree.

<Note>
  The main method includes a built-in example that finds the most recent common ancestor of **Bilbo** and **Frodo** from a Tolkien-style family tree file.
</Note>
